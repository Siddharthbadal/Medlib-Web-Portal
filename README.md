### Medlib Web portal

A portal to store patients records. Admin can read, add, update and delete these records.
Also has a search functionality with reset.


tech used

    - Django
    - Bootstrap
    - JavaScript
    - heroku for deployment 
    
  Live: https://medlibproject.herokuapp.com/
